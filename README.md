## 📖 프로젝트 소개

이 프로젝트는 사용자가 입력한 키워드를 기반으로 여러 쇼핑몰의 상품 정보를 수집하고  
이를 한 화면에서 확인할 수 있도록 제공하는 서비스를 구현한 프로젝트입니다.

상품 가격 비교 및 상품 정보 탐색을 보다 편리하게 할 수 있도록 하는 것을 목표로 합니다.

## 🔑 주요 기능

### 상품 검색
- 키워드를 입력하여 상품 정보를 검색

### 상품 목록 조회
- 검색 결과를 리스트 형태로 제공
- 가격 기준 정렬 가능

### 상품 상세 정보
- 상품 상세 정보 확인
- 원본 쇼핑몰 페이지 이동 가능
